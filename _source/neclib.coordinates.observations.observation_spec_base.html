classmethod from_file(file, /)[source]#

Read an observation specification from a TOML file.

This overrides Parameters.from_file() only for observation specs so that sky-coordinate fields can be interpreted with their coordinate-frame context. In particular, J2000/FK5/ICRS lambda_* fields represent RA. Therefore, sexagesimal RA strings such as "05:35:17.3" in "lambda_on[deg]" are normalized as hour angle before the generic unit parser sees them.
