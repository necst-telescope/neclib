Utility functions for arithmetic operations.

clip(value, minimum=None, maximum=None, *, absmax=None)[source]¶

Limit the value to the range [minimum, maximum].

Parameters

  • value (float) – Arbitrary parameter.

  • minimum (Optional[float]) – Lower limit of value.

  • maximum (Optional[float]) – Upper limit of value.

  • absmax (Optional[float]) – Upper limit of absolute value of value.

Return type

float

Examples

>>> clip(1.2, 0, 1)
1
>>> clip(41, 0, 100)
41
>>> clip(-4, absmax=3)
-3
frange(start, stop, step=1, *, inclusive=False)[source]¶

Float version of builtin range, with support for stop value inclusive.

Parameters

  • start (float) – First value to be yielded.

  • stop (float) – Last value to be yielded never exceeds this limit.

  • step (float) – Difference between successive 2 values to be yielded.

  • inclusive (bool) – If True, stop value can be yielded when stop - start is multiple of step.

Return type

Generator[float, None, None]

Notes

Because of floating point overflow, errors may appear when print-ing the result, but it’s the same as almost equivalent function numpy.arange.

Examples

>>> list(frange(0, 1, 0.2))
[0, 0.2, 0.4, 0.6, 0.8]
>>> list(frange(0, 1, 0.2, inclusive=True))
[0, 0.2, 0.4, 0.6, 0.8, 1]
